Creating a Conda env from a requirements.txt file

By 0x7df, Wed 26 August 2020, modified Wed 26 August 2020, in category Misc

TIL Anaconda Python

First create a new environment, including pip:

conda create --name <new_env> pip

Then you can do the standard installation from the pre-existing requirements.txt file:

conda activate <new_env>
pip install -r requirements.txt

(From Datumorphism.)
